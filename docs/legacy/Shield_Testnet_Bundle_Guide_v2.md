## Goal

This guide explains how DigiByte developers can run the Quantum Immune Shield
on testnet.

---

## Requirements

- Python 3.11
- access to `digibyted` RPC
- node and wallet endpoints

---

## Testnet Execution Plan

### Step 1 - Start the testnet node

Enable RPC, wallet, and logging.

### Step 2 - Connect bridges to live APIs

Replace mock code with actual:

- RPC calls;
- wallet API calls; and
- event listeners.

### Step 3 - Run the full pipeline

Stream live events from testnet into the Orchestrator.

### Step 4 - Review immune responses

Study how the Shield reacts to real chain behaviour.
